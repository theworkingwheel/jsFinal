Start Programming with Processing

In order to program your computer, you first need a specialized program that will translate your ideas in a language the computer understands. We will be using a program called Processing that allows us to create images and interactive animation with a computer. Once you completed the installation steps[1], go ahead and open the Processing program. Also, because the name of the program is “Processing”, which is a general term. I will abbreviate the name the way the authors of Processing do, which is "PDE" (Processing Development Environment)[2].

When you start PDE, you will see a blank window with a blinking cursor just like you would see in a Microsoft Office Word document. There is also a toolbar at the top of the window. The two buttons on the left look like media playback controls. Lets start writing a simple line of code instruction in order to see what the buttons do. [2]

Type in the following below or Copy & Paste into the PDE window:

Copy & Paste Code 1

rect(50,50,30,30); [3]

Click on the button that looks like the Play media symbol, which is the “Run” command in the upper-left corner of the PDE window. Another window should appear with a square like the one below.

The window above is called a “sketch” in the PDE. To close the sketch, click on the “Stop” button which looks like a stop playback control.
